Setup instructions

  • Windows
  • Linux
  • Mac

Windows

Vedran Load Balancer

  • 1. Download latest release from: Github
  • 2.Create polkadot account - wiki.polkadot.network
  • 3.
    ./vedran start --auth-secret=<generate random secret> --private-key=<private seed from generated account>
  • 4. Use --help for additional configuration and flags

Vedran Daemon

  • 1. Download latest release: Github
  • 2.
    ./vedran-daemon --id=<some secret id for your node> --lb=<load balancer url> --payout-address=<polkadot account address>
