Setup instructions

Help us decentralize infrastructure by either hosting public load balanacer or joining existing one as node operator.

  • Windows
  • Linux
  • Mac

Vedran Load Balancer

Use this for hosting public endpoint and distributing incomming requests amongst nodes in it's pool. You will also be in charge of securing public endpoint and finding source of funding (ideally chain treasury) to distribute amongst nodes in pool.
  • 1.Create polkadot account - wiki.polkadot.network
  • 2.Download latest release from: Github
  • 3.Unzip downloaded archive with your favorite tool
  • 4.
    ./vedran start --auth-secret=<random secret> --private-key=<private seed from step 1.>
  • 4. Use --help for additional configuration and flags

Vedran Daemon

Ran this alongside your Substrate based node to join to some existing load balancer pool.
  • 1. Run Substrate based node: wiki.polkadot.network
  • 2. Download latest release: Github
  • 3.Unzip downloaded archive with your favorite tool
  • 4.
    ./vedran-daemon --id=<some secret id for your node> --lb=<load balancer url> --payout-address=<polkadot account address>
