Setup instructions

Help us decentralize infrastructure by either hosting public load balancer or joining the existing one as a node operator.

  • Windows
  • Linux
  • Mac

Vedran Load Balancer

Vedran load balancer is providing public endpoint for end users, distributing request amongst available nodes and doing recurring payments to node operators based on various metrics and number of requests served.

Note: you will have to secure funding yourself either trough chain treasuries or other sources of funding
  • 1.Create polkadot account - wiki.polkadot.network
  • 2.Download latest release from: Github
  • 3.Unzip downloaded archive with your favorite tool
  • 4.
    ./vedran start --auth-secret=<random secret> --private-key=<private seed from step 1.>
  • 4. Use --help for additional configuration and flags

Vedran Daemon

Vedran daemon is small daemon app that is running alongside your Substrate based node to join some of the existing load balancer pools and receive payments for operating node.
  • 1. Run Substrate based node: wiki.polkadot.network
  • 2. Download latest release: Github
  • 3.Unzip downloaded archive with your favorite tool
  • 4.
    ./vedran-daemon --id=<some secret id for your node> --lb=<load balancer url> --payout-address=<polkadot account address>
