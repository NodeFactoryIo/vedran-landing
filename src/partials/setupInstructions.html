Setup instructions

  • Windows
  • Linux
  • Mac

Windows

Load balancer

  • Download latest release: github.com/NodeFactoryIo/vedran/releases
  • Create polkadot account - wiki.polkadot.network/docs/en/learn-account-generation
  • ./vedran start --auth-secret=<generate random secret> --private-key=<private seed from generated account>
  • Use --help for additional configuration and flags

Node

  • Download latest release: github.com/NodeFactoryIo/vedran-daemon/releases
  • ./vedran-daemon --id=<some secret id for your node> --lb=<load balancer url>
    --payout-address=<polkadot account address>
