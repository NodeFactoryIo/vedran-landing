Vedran

Auto-funded public P2P infrastructure
intended for Substrate based chains.

Features

Decentralized

Stop relying on cloud infrastructure

Autofunded

Get paid for providing nodes

Multichain

Supports all substrate based chains but it can be modified to support other blockchains

How it works

Vedran creates a tunnel between load balancer and regular Polkadot nodes which allows nodes to serve requests from home networks while getting paid based on how useful they are.
