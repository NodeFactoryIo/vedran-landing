Vedran

Auto-funded public P2P infrastructure
Intended for Polkadot chains, specifically Kusama.

Features

Decentralized

Stop relying on cloud servers

Autofunded

Run node and earn money

Multichain

Currently supports all substrate based chains but it can be modified to support others

How it works

Vedran creates a tunnel between load balancer and regular Polkadot nodes which allows nodes to serve requests from home networks while getting paid based on how useful they are.
